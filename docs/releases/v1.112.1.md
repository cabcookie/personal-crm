# Release: 1.112.1: Copy Project Notes

## Major Changes

Added a new feature to copy project notes to clipboard from the project notes view. Users can now quickly copy notes from the last 1, 2, or 4 weeks with a single click. The feature includes:

- Three buttons for different time ranges (1, 2, or 4 weeks)
- Loading state with spinner while fetching notes
- Confirmation message that smoothly fades out over 1 second
- Notes are formatted as markdown with dates and meeting context

## Fixes

- Sync `package.json` and `package-lock.json`

## Details

### UI

- Added `CopyProjectNotesButtons` component with three preset time range buttons
- Loading state displays spinner and "Copying notes..." message
- Success state shows checkmark with confirmation that fades from muted foreground to white over 1 second
- Component automatically resets after 2 seconds

### API

- New `copyProjectNotes` function fetches project activities from specified time range
- Queries project activities via GraphQL with comprehensive selection set including note blocks, todos, and meeting context
- Converts notes to markdown format with proper date headers
- Sorts notes by date (most recent first)
- Copies formatted markdown to system clipboard
