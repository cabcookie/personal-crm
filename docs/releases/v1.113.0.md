# Release: v1.113.0: REST API for External Integrations

## Major Changes

Added a comprehensive REST API infrastructure to enable external integrations to access Personal CRM data. This release introduces:

- API key management system for creating and managing authentication credentials
- REST API backend with secure authentication middleware
- GraphQL query integration to serve data through REST endpoints
- Support for retrieving accounts, projects, people, and meeting data

## Details

### API Key Management

- Created API key generation and storage system
- API keys are securely stored in DynamoDB with user ownership
- Support for creating, listing, and revoking API keys
- Each API key is associated with specific user permissions

### Data Integration

- Connected REST API to existing GraphQL backend
- Supports fetching accounts, projects, and people
- Returns data in JSON format optimized for external consumption
- Maintains existing authorization and data access patterns

### Smaller Changes

- Updated dependencies to ensure compatibility
- Synchronized package.json with package-lock.json
